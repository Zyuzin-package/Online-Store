Доступно для всех пользователей
/ - главная страница
/category/{category-name}
/product/{product-name}| Нужно залочить инпут для оставления отзыва для неавторизированных пользователей
/login
/registration

Доступно для авторизированных пользователей
/my/main - профиль пользователя | Нужно сделать отдельную страницу, как раз main, и выводить туда данные о активных заказах и просьбы оценить товары, с возможностью выставлять звезды и кнопкой не оценивать
/my/settings - изменение профиля пользователя
/my/orderlist - страница с заказами.
/my/cart - корзина 

Доступно для руководящий ролей
/admin/users/ - Все пользователи
/admin/users/edit - изменение роли юзера | Попробовать запретить менеджерам выставлять роли админов для юзеров и изменять роли админов
/admin/users/orders - Список всех заказов пользователей
/admin/users/orders/{status} - Список заказов с определенным статусом
/admin/product/new - Создание нового продукта
/admin/product/edit/{product-title} - Изменение продукта
/admin/product/category - Вывод всех категорий
/admin/product/category/edit/{category-name} - Изменение категории
/admin/product/category/new - Создание категории

OrderStatus:
СОЗДАНО
Заказ оплачен и ожидает обработки
ПРИНЯТО В РАБОТУ
Заказ оплачен. На данном этапе производится комплектация и упаковка товаров
ПЕРЕДАНО В ДОСТАВКУ
На данном этапе происходит транспортировка товаров до пункта выдачи
ОЖИДАЕТ В ПУНКТЕ ВЫДАЧИ
Срок хранения товаров 5 дней
ПОЛУЧЕНО
